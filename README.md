# TG-BOT
# SWILL CASINO — два бота для казино

## Команды игрового бота:
- `/start` — регистрация
- `/play` — выбрать игру
- `/stats` — твоя статистика
- `/stats_player @username` — статистика другого
- `/top money` или `/top level` — топ-50

## Команды админ-бота:
- `/start` — открыть админ-панель
- Кнопки для управления игроками и промокодами

## Игры:
- 🔫 Русская рулетка (x5)
- 🔴 Красное / Чёрное (до 12 игроков)
- 🎲 Кости (2 кубика)
